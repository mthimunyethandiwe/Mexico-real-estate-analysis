# Mexico Real Estate Analysis

## Research Question
> Are property prices in Mexico more influenced by property size or by location?

## Overview
An exploratory data analysis of 1,736 Mexican property listings 
using Python, Pandas, Matplotlib, and Seaborn.

## Project Status
🚧 Work in Progress — Lesson 4 coming soon

## Notebooks
- `01_data_cleaning.ipynb` — Loading, cleaning and combining raw data
- `02_visualization.ipynb` — Histograms, boxplots, scatterplots, small multiples
- `03_correlation.ipynb` — Pearson correlation, heatmap, Simpson's Paradox

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Data
Mexican real estate listings data provided as part of 
WorldQuant University Applied Data Science Lab.
